Lapisan *aplication* model OSI adalah lapisan yang paling Anda kenal. Keakraban ini karena lapisan aplikasi adalah lapisan di mana protokol dan aturan ditempatkan untuk menentukan bagaimana pengguna harus berinteraksi dengan data yang dikirim atau diterima.

Aplikasi sehari-hari seperti email, browser, atau *software* penjelajahan server file seperti *FileZilla* menyediakan *Graphical User Interface* (GUI) yang ramah bagi pengguna untuk berinteraksi dengan data yang dikirim atau diterima. Protokol lain termasuk DNS (*Domain Name System*), yaitu bagaimana alamat situs web diterjemahkan ke dalam alamat IP.
